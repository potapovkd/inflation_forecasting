# Прогнозирование инфляции в России
## Студенческая научная работа
Исследование возможностей применения моделей машинного обучения для краткосрочного прогнозирования инфляции.

Апробирование моделей множественной линейной регрессии (в т.ч. с L1-регуляризацией) и ансамблевого алгоритма градиентного бустинга над решающими деревьями.

Адаптация модели к прогнозированию по меньшему числу факторов-признаков.
